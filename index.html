Ming Commands Guide

This is a WIP so things might change
Required arguments are in [square brackets],
Optional arguments are in {curly brackets}.

Commands

friend friendreq - [DOTA Only] The bot sends you a friend request, it wont it's already in your friend list.

countdown cd - [DOTA Only]

discord - Gives you the discord invite link.

mmstats {region} - Players searching in xx region.

notes - Important notes an admin might have left, be sure to check it regularly!

admins - Get a list of admins

stats - not implemented yet

restart - Admin only

admin [command] [argument] - Admin only
